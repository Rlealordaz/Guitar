### Installation

1. Clone the repository
```bash
git clone <your-repo-url>
cd ts-guitar-por-mi
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
